fetch - функция в JS для отправки запросов

```js
// Самый простой get-запрос
// Подключаемся по fetch, нам приходит какая-то информация
// нам нужно с ней что-то сделать, делаем первым .then
// обзываем данные как угодно (data) и смотрим что это
// это скорее всего промис. нам нужно его сериализировать
// в json формат, так что пусть функция вернёт нам return data.json()
// делаем второй .then сразу после первого, привязывая его к нему,
// и обозвав то что у нас теперь в виде json как info, выводим в консоль

const data = fetch(
    'https://jsonplaceholder.typicode.com/users/10'
).then((data) => {
    console.log(data)
    return data.json();
}).then((info) => {
    console.log(info);
})
```
